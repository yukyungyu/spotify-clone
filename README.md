# 프로젝트 소개
Spotify API를 이용하여 저희만의 방식대로 만든 스포티파이 클론 사이트입니다.

## 개발 기간
* 24.07.30 ~
회사 프로젝트 일정이 여유로울 때마다 틈틈히 만들었습니다.

## 멤버 구성
- 조미혜 - 로그인 기능 구현, 메인 페이지, 앨범, 아티스트 컴포넌트
- 김유경 - 로그인 기능 구현, 검색 페이지, 플레이리스트 컴포넌트
